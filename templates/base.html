Prediksi Nilai Tukar Bitcoin dan US Dollar dengan Metode Long Short-Term Memory

Pelatihan Model Hasil Prediksi

Pendahuluan

Nilai Bitcoin sangat volatile. Untuk itu, perlu dibuat sebuah model untuk memprediksi pergerakan harga bitcoin menggunakan machine learning. Machine Learning merupakan salah satu cabang dari AI yang mencapkup perancangan algoritma yang memungkinkan komputer memiliki kemampuan untuk belajar dari data, tanpa secara eksplisit harus mengikuti instruksi yang diprogram oleh pembuat sistem (Budiharto, 2016). Salah satu metode yang bisa digunakan adalah Long Short-Term Memory atau LSTM. Long Short-Term Memory (LSTM) adalah salah satu cabang dari Recurrent Neural Network (RNN) yang sangat cocok diterapkan pada dataset berbasis time-series dan masalah jangka panjang.
